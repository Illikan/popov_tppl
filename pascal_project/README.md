Интерпретатор упрощенного языка Pascal.

### Запуск
Запуск примеров из папки `examples`:
```bash
lua pascal/main.lua examples/ex2.pas
```
Программа выведет итоговые значения всех переменных.

### Автотесты
Нужен установленный `busted`.

Прогон тестов:
```bash
busted
```

Проверка покрытия (100%):
```bash
busted --coverage
luacov
```
